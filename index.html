Hello world! I’m Sarah and I’m a front-end web developer from Canton, Ohio. I create the front-end elements for web applications and translate designs into functional websites. I build WordPress websites for small businesses. I love build with JavaScript, NodeJs, and MongoDB. In addition to building and testing web applications, I have 9 years of experience teaching, designing curriculum, and differentiating instructional materials for different learning styles. I am looking for a full-time remote front-end web developer role.

I design the overall plan and look for web applications. I collaborate with clients on the user experience for websites. I work out sitemaps, user personas, and wireframes so designs will work on the web. I organize and plan the vision for a website including the fonts, colors, and icons. I use design tools such as Adobe Creative Suite, Canva, and Visme to make web graphics for websites.

I convert design comps into live responsive websites accessible for all users. I build everything users see on a website such as web pages, menus, forms, and newsletters. I also edit and customize existing websites so they stay up to date with current coding standards. I test, debug, and maintain web applications regularly so they run smoothly on the web. I write clean, semantic, well formatted code so other developers can easily access and use with.

I construct and maintain the behind the scenes structure for websites. I set up databases for websites with MongoDB. I have experience using programming languages such as PHP, Ruby, and Python to help databases connect with servers. I particularly enjoy using NodeJs to generate dynamic web applications.
