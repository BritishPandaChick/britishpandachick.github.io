About Sarah

Hello world! I’m Sarah and I am a frontend web developer from Canton, Ohio. I design, build, and edit WordPress websites for small businesses and creatives. I also have experience working on various code related projects using Node.js, React, Ruby, and Python.

I have completed two Skillcrush career blueprints where I learned how to code with JavaScript, Ruby, PHP, and WordPress. Recently I learned how to use Node.js, Express.js, and MongoDB to create web applications. In 2021 I learned how to use Node.js to build Alexa skills and learned how to use Sass to compile styles on a website. I love learning and I’m currently learning how to use Ruby on Rails to create web applications.

In addition to building websites, I am also a producer at BritishPandaChick. I originally started blogging on The Original BritishPandaChick blog where I created the My Coding Journey series. I spent two years managing the BritishPandaChick Codes blog where I talked about coding, careers, and books. I also managed Needlework Kits blog where I talked about craft kits I made. Now I write posts on The Practical Dev and Code Newbie about learning how to code and web design. I have written content for other platforms including App Academy Book Prescriber.

Before I started learning how to code, I spent 9 years teaching students academic skills at different schools in Stark County and at Sylvan Learning. Today I combined my coding skills and teaching background together to mentor new developers just getting started in tech. In April 2020, I taught a brown bag lesson to Lambda School students on how to use Bootstrap to create a portfolio website. In October 2021, I gave a presentation on how to start freelancing to Girl Develop It students.

I served as the moderator for the Elphas Can Code online community on Elpha from 2019 - 2022. In July 2019, I created the Disney Codes Challenge in order to help developers develop the habit of building more projects. I also mentor new developers in my spare time by providing support and empowering developers as they continue to code. In 2020, I became an VIP expert on Fairy God Boss and started sharing advice to help job seekers with their careers.
