About Sarah

Learning tech skills has been one of the most powerful defining moments in my life. I started learning how to code in August 2015 after spending years in teaching. I wrote my first line of code that summer. Although it was just "Hello World", that line of code transformed my life and started me on the path to become a front-end developer.

My coding journey has its own share of ups and downs. Every time I open my text editor, I'm always learning something new that makes me a better with every challenge that pop up in my code. Now I'm finding ways to fuse my tech skills with my teaching experience starting with the series My Coding Journey on my blog The Original BritishPandaChick where I review all the tech skills I've learned on my own journey.

In 2017, I launched my second blog Needlework Kits which reviews needlecraft kits I've make in my spare time. I have been using everything I've learned about UX design to plan the user experience for my readers. I have been using my new tech skills to help others looking to get started on the web from building websites to mentoring other developers on how to get started in tech. I formed a mastermind group with 3 other Skillcrush alumni with a mission to inspire and motivate women to do the best work possible.
