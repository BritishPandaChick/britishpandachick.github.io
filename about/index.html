About Sarah

Sarah Bartley is a front-end web developer and content creator at BritishPandaChick in Canton, Ohio. She consults with small business, creatives, and solopreneurs to build WordPress websites. Before she was composing web applications, she taught academic skills to students and created instructional materials differentiated for individual student learning needs.

In addition to producing websites, Sarah writes posts for her blogs BritishPandaChick Codes and Needlework Kits. She has contributed her posts to other platforms such as The Practical Dev, App Academy, and Book Prescriber. She has been featuredSarah currently serves as the moderator for Elphas Can Code community on Elpha.

Elphas Can Code

In September 2019, I created the Elphas Can Code community on Elpha, an online platform for women in tech to network and talk. Elphas Can Code is a supportive community for all the Elphas learning how to code. This community is meant to help all women in tech learn coding concepts. Elphas Can Code welcome all women in tech from newbies getting started learning how to code to experienced developers learning coding concepts for the job. I serve as the community moderator where I review posts, enforce Elpha guidelines, and creating conversations for all members to participate.
