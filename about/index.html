About Sarah

Sarah Bartley is a front-end web developer and content creator in Canton, Ohio. Sarah builds websites for small businesses using JavaScript, PHP, and WordPress. Before she was making web applications, she spent 9 years teaching academic skills to students and creating differentiated instructional materials for individual learning needs.

When she's not coding, Sarah manages the blogs BritishPandaChick Codes and Needlework Kits. She has contributed posts to other platforms such as The Practical Dev, App Academy, and Book Prescriber. Sarah serves as the moderator for the Elphas Can Code online community on Elpha.

Disney Codes Challenge

Founded July 2019

Disney Codes Challenge is a 57-day coding challenge. Developers are committed to building web projects inspired by Disney properties each day for 23 minutes. Developers document their progress the challenge with the journal from the challenge repository on Github.

Elphas Can Code

Founded September 2019

Elphas Can Code is a supportive online community for all Elpha members learning how to code. The community strives to help all members at different stages of their coding journeys. I serve as the community moderator where I review posts, enforce Elpha guidelines, and create conversations so all members can contribute.
