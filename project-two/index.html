Magic 8 Ball

Objectives

  • Create an object for magic 8 ball and array property for answers.
  • Create a method to get question, shake the ball, and display answer.
  • Hide magic 8 ball answer and allow answer to fade in.
  • Swap images when trying to reveal answer.
  • Debug code files.

Background

I have made two versions of this project throughout my coding journey. The first version of the magic 8 ball was a form where users typed in their questions and had their answers displayed on the page. Behind the scenes was jQuery working to get a random answer and fading it onto the page.

I revisited this project a couple of years later and made changes to the look of the page as well as the code. Before I started writing any code, I took a few minutes to write some pseudo code. The pseudo code broke down the specific steps the code needed to do so the magic 8 ball would work. This pseudo code would serve as the base of the onClick method.

First, I created a magic 8 ball object and set up an array property with an array of answers. Next, I made the askQuestion method that returns a random answer when I ask a question. I finally used some jQuery to switch out magic 8 ball images, hide text, and get answers to fade in and out on the magic 8 ball. The last thing that needed to be done was debugging the entire site and fixing any errors in this code.
