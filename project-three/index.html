Election Map

Objectives

  • Create a factory function to create candidates
  • Create an array of election results. Update array with recounts.
  • Tallying up results
  • Assign winners to each state
  • Populate table with winners

Background

The focus on this project was mostly to get the functionality set up and working. First, I created a politician object and properties. Next, I made the createPolitician function and put the politician object and properties I just made inside of it. Then I created two new variables for the candidates. Inside these new variables were new instances of the createPolitician function that set the property name. I created two arrays of election results for each of the candidates and updated the arrays with the recount information.

Second, I created two methods to tally up votes and declare the winner. The first method I created tallyUpTotalVotes(). This method uses a for loop to go through all the electionResults and add up all the votes. I set the second method to the variable winner. Inside this method is an if-else statement that picks the candidateâ€™s name based on who had the most totalVotes. I made the setStateResults function that set the results for each state and would change the color of each state on the map once a winner was declared.

Finally, it was time to tackle the table and get information to appear on the table to show the final results. I created the countryInfoTable variable that grabbed the countryResults id in the HTML file. I began making lots of variables for either positions of the table and ones for displaying the text. Initially I put some of these variables at the bottom of my code. This resulted in issues with the table data. However, after some debugging and testing I realized everything worked once it was put inside the setStateResults function.
