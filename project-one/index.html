Duchess of Cambridge Tribute Page

Objectives

  • Have a main section for all web page elements.
  • Have a title element that describes the subject of the page.
  • Have an image on the page with a caption underneath describing the image. The image should responsively resize and be centered with the parent element.
  • Have section for tribute information that describes the subject of the page.
  • Have a link that goes to an outside website with more information about the subject of the tribute page.

Background

The original version of the tribute page was made in 2016. It was one of the first projects I made using Bootstrap. At the time I made this project, I used Bootstrap 3. Over the years, I have updated this project from Bootstrap 3 to the current version Bootstrap 4.

Although there is some CSS on the page, the focus is mostly on Bootstrap. I wanted to let Bootstrap be the star on this site so I kept things very simple in regards to the styling and instead spent my time concentrating on the page layout. In the original version, I used a lot of div tags with Bootstrap classes to get the elements centered on the page or inside rows. I even wrapped some of the elements in div tags to help describe what certain elements were on the page.

After I finished the tribute page, I have made several updates to the HTML and Bootstrap classes over the years. I replaced the div tags with header, main, and figure tags to make my code more semantic. I removed the older Bootstrap classes and replaced these classes with ones that work with Bootstrap 4.
